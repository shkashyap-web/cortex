# Project Overview

CORTEX is an AI-native Enterprise Banking Operating System designed for large financial institutions such as IDBI Bank.

It is NOT a chatbot.

It is NOT a dashboard.

It is NOT a collection of independent AI models.

CORTEX is a Decision Intelligence Platform that continuously understands customers, MSMEs, branches, portfolios, risks, fraud, compliance, operations, and executive objectives, and coordinates intelligent decisions across the enterprise.

Think of it as the enterprise cognitive layer that sits above existing banking systems.

---

# Vision

Build the operating system that powers every important banking decision.

Every customer interaction, transaction, recommendation, fraud alert, lending decision, investment recommendation, MSME assessment, and executive insight should flow through CORTEX Decision Engine (CORTEX DE).

# Product Philosophy

CORTEX is enterprise software.

Do NOT build hackathon shortcuts.

Every component should be modular.

Everything should be replaceable.

Every feature must communicate through interfaces.

Avoid tight coupling.

Never write code that assumes only one bank.

Everything should be configurable.

---

# Architectural Principles

1. Event Driven

Modules communicate through Enterprise Event Bus.

Never call unrelated services directly.

---

2. Explainable AI

- evidence
- confidence
- alternatives
- audit trail

3. Enterprise First
